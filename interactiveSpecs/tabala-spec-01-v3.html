Event list — card fields
  • 1
    Thumbnail photo — 64×64px rounded. Emoji placeholder when no image. Tappable: opens detail.
  • 2
    Event name — bold, max 2 lines, ellipsis. Primary identifier.
  • 3
    Location — pin icon + venue name. 1 line truncated.
  • 4
    Date & time — calendar icon + "DD de mes, YYYY · HH:MM". Always visible.
  • 5
    Confirmed count — people icon + "N confirmades" in red. No RSVP controls on list.
  • 6
    Chevron › — full row is the tap target. Opens detail.
Event detail — screen sections
  • 1
    Hero image — full-width, 160px tall. Dark gradient fallback. Top of screen.
  • 2
    Title + "Confirmada" badge — green pill shown only after RSVP confirmed. Top-right of title.
  • 3
    Location / Date / Confirmats — icon rows; confirmats count in red. Below title.
  • 4
    Description — free text from Mestre. Greyed, scrolls if long.
  • 5
    CTA "Confirmar Assistència" — full-width red button. Turns dark green when confirmed. Only here, not on list.
  • 6
    Photo gallery — 2-col grid + "Afegir" button. Visible on past events only.
RSVP behaviour
✓ Confirmed
Button turns dark green. "Confirmada" badge on title. Added to confirmats count.
Not confirmed
Button stays red. No badge. Not counted. Reversible at any time before event.
RSVP location
Detail screen only. The list is read-only — shows counts but has no action buttons.
Persistence
Saved to server on tap. Survives app close and reopen.
Past events
CTA replaced by the photo gallery section after the event date/time passes.
Top bar & navigation
Top bar (list)
Group name + city left. Circular group logo right. Dark red background.
Top bar (detail)
Back arrow + "Detalls de l'esdeveniment" + share icon. Same dark red.
Bottom nav
4 tabs: Inici (active), Muixerangues, Figures, El meu perfil. Active = red icon + red label.
Edge cases
Behaviour rules for boundary scenarios.
Empty list
Empty state + "No upcoming events." CTA to view past events if any exist.
No hero photo
Group colour placeholder with event name initial. No broken image icon shown.
Long event name
List: 2 lines max + ellipsis. Detail: wraps up to 2 lines below hero.
Network error on RSVP
Button reverts to previous state. Toast: "Could not save. Check connection."
0 confirmats
Shows "0 confirmades". Row not hidden. No special styling.
Today's event
Date shows "Avui · HH:MM" instead of full date. Subtle accent highlight on card.
